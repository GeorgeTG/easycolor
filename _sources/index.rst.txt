:mod:`easycolor` --- Welcome to easycolor's documentation!
==========================================================

The package contains 2 modules that handle text styling.
The :mod:`easycolor.style` module provides
simple and re-usable styling.
The :mod:`easycolor.parser` module provides
advanced styling that is also re-usable.
